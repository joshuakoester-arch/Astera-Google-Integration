Privacy Policy

Overview

This application is an internal integration used by etrailer to access Google Sheets data for operational and reporting purposes.

The application is not a public service and is not intended for use by the general public.

Data Access

The application accesses Google Sheets data only after explicit authorization by a Google account holder through Google OAuth 2.0.

The data accessed is strictly limited to the scopes approved during the OAuth consent process and is used solely to support internal extract, transform, and load (ETL) workflows.

Data Storage

The application does not permanently store Google user data outside of internal processing systems required for business operations.

No Google user data is sold, shared, or distributed to third parties.

User Access

Only authorized internal personnel may use this application. There is no public sign-up, registration, or onboarding process.

Security

All access to Google APIs is performed over secure, encrypted connections using industry-standard OAuth 2.0 authentication mechanisms.
