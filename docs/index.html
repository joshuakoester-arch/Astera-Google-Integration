Astera Google Integration

This site hosts the public pages required for the Astera Google Integration OAuth consent screen.

Overview

The Astera Google Integration is an internal tool used by etrailer to securely access Google Sheets data via OAuth 2.0 for internal extract, transform, and load (ETL) workflows.

This is not a public application and is not intended for use outside of etrailer.
