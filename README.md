# Astera Google Integration

## Overview

This repository contains documentation and public-facing policy pages for the **Astera Google Integration**, an internal tool used by etrailer.

The integration is designed to securely access Google Sheets data via OAuth 2.0 for internal extract, transform, and load (ETL) workflows. It is not a public application and is not intended for use outside of etrailer.

## Purpose

- Read Google Sheets data for internal reporting and operational workflows
- Support automated ETL processes
- Operate exclusively for internal business use

There is no public sign-up, onboarding process, or external access.

## Status

This repository exists to support Google OAuth consent and verification requirements by providing a public application homepage, privacy policy, and terms of service.
